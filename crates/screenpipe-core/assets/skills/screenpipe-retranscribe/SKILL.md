# Screenpipe Retranscribe

This skill enables Pi to retranscribe audio recordings with different models or
settings.

## Usage

Use this skill when the user wants to:

- Improve transcription accuracy
- Use different language models
- Adjust speech recognition settings
- Reprocess old recordings

## Capabilities

- Multiple transcription engines
- Language selection
- Diarization (speaker separation)
- Custom vocabulary

## Guardrails

- Only retranscribe recordings the user owns
- Consider processing time for large files
- Handle unavailable models gracefully
