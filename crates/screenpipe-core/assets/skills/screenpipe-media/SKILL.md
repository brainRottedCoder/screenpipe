# Screenpipe Media

This skill enables Pi to export video and audio recordings captured by
Screenpipe.

## Usage

Use this skill when the user wants to:

- Export screen recordings as video
- Export audio recordings as audio files
- Create clips from recordings
- Generate GIFs from video

## Capabilities

- Video export (MP4, GIF)
- Audio export (MP3, WAV)
- Clip generation with time ranges
- Quality/format options

## Guardrails

- Only export recordings the user owns
- Respect storage limits
- Confirm before large exports
