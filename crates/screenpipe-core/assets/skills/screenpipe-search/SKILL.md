# Screenpipe Search

This skill enables Pi to search through all screen, audio, and UI data captured
by Screenpipe.

## Usage

Use this skill when the user wants to:

- Find specific content on their screen
- Search through recorded audio/transcriptions
- Query historical screen captures
- Find UI elements by text or properties

## Capabilities

- Full-text search across all frames
- Time-range filtering
- App/window filtering
- OCR text search
- Audio transcription search

## Guardrails

- Only search data that the user has captured
- Respect privacy settings
- Confirm before showing sensitive content
